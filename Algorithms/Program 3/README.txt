## Program3 (Renee Ti Chou)

The source code for this program was built using Eclipse, and it can be compiled by JDK 8.


## Usage

java Program3 [input file path][output file path]


## Source Files

1. JobPlanner.java
2. Program3.java


## Input Format

Low stress and high stress jobs are in two separate lines starting with “L” and “H”
respectively. Data should be space-delimited or tab-delimited. e.g.

L 10 1 10 10 
H 5 50 5 1 
L 10 10 1 10 
H 5 30 100 20 
...
